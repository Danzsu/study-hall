4. **Az alábbiak közül melyik követelményt kell egy modern kriptográfiai hash függvénynek kielégítenie?**
    1. Második őskép ellenállóság (second preimage resistance): egy $y$ hash értékhez legyen nehéz olyan $x$ bemenetet találni, melyre $H(H(x))=y$
    2. Gyenge ütközés ellenállóság (weak collision resistance): egy adott $x$ bemenethez legyen nehéz egy másik $x'$ bemenetet találni, melyre $H(x)=H(x')$
    3. Erős ütközés ellenállóság (strong collision resistance): legyen nehéz olyan $x$ és $x'$ különböző bemeneteket találni, melyekre $H(x)=H(x')$
    4. Őskép ellenállóság (preimage resistance): egy adott $y$ hash értékhez legyen nehéz olyan $x$ bemenetet találni, melyre $H(x)=y$.
