## Fo pontok

- Where on the stack are function parameters and local variables placed?A stack frame is a memory structure created during a function call to store its local variables, parameters, and return address.
- Local variables are placed below the saved EBP register at EBP - offset .
- Function parameters are placed above the saved EBP register at EBP + 8, EBP + 12, etc., depending on the calling convention .
- What is the main idea of stack overflow?Stack overflow occurs when a program writes more data to a buffer on the stack than it can hold, overwriting adjacent memory, including the return address.
- This allows attackers to hijack the program's control flow and execute arbitrary code.
- Where can the attacker's code be injected in a stack overflow attack?The attacker's code shellcode can be injected: Into the local buffer itself, often preceded by a NOP sled.
